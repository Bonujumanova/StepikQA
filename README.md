# StepikQA
Exercices from Stepik
Итоговое тестовое задание.

1.Создать проект

2.Выбрать интернет-магазин для тестирования - НЕ ИСПОЛЬЗОВАТЬ УЧЕБНЫЕ САЙТЫ!!! Не берите ДНС, Яндекс, Мвидео - трудности с запуском теста, регистрацией, локаторами!

Вручную пройдите весь сценарий, убедитесь что нет всплывающих уведомлений, не требуется авторизация по смс и т.д. Попробуйте его запустить и убедитесь что он нормально запускается и только тогда приступите к написанию тестового проекта

3.с помощью фильтров выбрать товар (чем больше фильтров тем лучше)

4.добавить товар в корзину и пройти весь бизнес путь

5.Использовать в построении проекта и теста принципы ООП и Page Object Model и PyTest

6.Максимально усложнить бизнес логику теста, для закрепления знаний, использовать проверочные методы

7. Написать подробную аннотацию к проекту
